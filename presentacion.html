Inteligencia Financiera

El sistema protege la rentabilidad aplicando lógica avanzada según el tipo de contrato.

  • Fijo por Servicio: Alerta si el margen es < 10% o si hay pérdida directa.
  • Excepciones Geográficas: Ciudades estratégicas (ej. Lima, B. Aires) exentas de regla de margen.
  • Booking / I Need Tours: Permite "pérdida controlada" de hasta $5.000 en contratos variables.
  • Validación TC: Control automático de Tipo de Cambio para ciudades clave.
NUEVA FUNCIONALIDAD

Control de Riesgo en Efectivo

Detectamos automáticamente situaciones de alto riesgo operativo y financiero.

  • Si el convenio es "PARTICULARES SIN CONVENIO"...
  • ...y el Medio de Pago es "EFECTIVO".
  • Acción: El sistema marca inmediatamente el registro como DISCREPANCIA para auditoría manual obligatoria.

Integridad de Datos

Aseguramos que la información operativa esté completa antes de procesar pagos.

  • Centros de Costo (CC): Obligatorio para cuentas corporativas (Unilever, Godrej, etc.). Prohibido usar "PENDIENTE".
  • Travel Security: Validación cruzada especial entre CC y Naturaleza del Gasto.
  • Móviles Restringidos: Bloqueo automático de móviles de prueba (000, 100, 200, 300).
  • Campos Vacíos: Alerta si "Obs. Conductor" tiene datos indebidos.
  • Validación de Montos: El Costo y el Total deben ser mayores a $0.
